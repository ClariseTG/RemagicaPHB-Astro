# Astro PHB
A website written using astro.build for the Remagica 4e tabletop RPG system. This website is the sole official source of information on the TTRPG.

This is a port of [the PHB written in Twine,](https://github.com/ClariseTG/RemagicaPHB-Twine) and will hopefully eventually overtake the Twine version. I love Twine, but this project outgrew it years ago. Using a static site generator makes this project more compatible with Git and fixes some issues with the Twine version.

One of my major goals with this version of the site is to leverage its web-app-ness for interesting ways of presenting the TTRPG's information. We'll see how that goes; this is a hobby project.

# Remagica 4e
Remagica 4e is a TTRPG by ClariseTG (me.) It is a fantasy TTRPG tailored to my worldbuilding project by the same name, with a magic system hard enough that it's practically Sci-Fi. It is my fourth major attempt at codifying the rules of Remagica's world as TTRPG mechanics (hence 4e.)

All Remagica TTRPG systems are open for use in any way you want! If you play this game and have house rules you end up using, let me know in the Git Issues and I may integrate your fixes into the main game (with credit!) If you create your own world that adheres to this system's rules, I'd love to hear about it; you can find me on Tumblr as ClariseTG.
